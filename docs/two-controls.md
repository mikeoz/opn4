# The Two Controls

My data + Your AI + My control = Living Intelligence

Every person who uses Opn.li Agent Safe has two points of control
over every AI agent that touches their data.

## Control 1 — The Front Door (Verification Endpoint)

Every agent must check here before reaching your data.
The door asks: Is this agent registered? Does it have a permission slip?
If no — the door stays closed.
If yes — the agent gets exactly what the permission says. Nothing more.

## Control 2 — The Permission Slip (CARD)

A CARD — Community Approved Reliable Data — is the permission slip you write.
You say: this agent, this data, this action, this long, these rules.
You can revoke it any time. Immediately.

## Together

The Front Door controls access.
The CARD controls use.
Together: My data. Your AI. My control.
